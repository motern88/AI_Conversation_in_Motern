数据总览如下：

```python
└──video                          # 下载的包含一个视频的总文件夹
    └──images                     # 存放视频帧的文件夹
        ├──frame_0001.jpg
        ├──frame_0002.jpg
        ...
        └──frame_XXXX.jpg
```

下载该数据的名为`video`的文件夹，包含该视频的视频帧。在该视频对应的文件夹下：

- `images`文件夹：

  存放该条视频的所有视频帧，以 `.jpg` 格式存储；以 `frame_{四位数帧索引}.jpg` 命名。
